# Java GS VittaTech

Este é um projeto Java criado como parte dos estudos na faculdade, focado em aplicar os
principais conceitos da linguagem como lógica, orientação a objetos, condicionais e laços.

## 🛠️ Tecnologias

- Java

## 📌 Objetivo

Consolidar aprendizado de Java por meio de exercícios práticos e código organizado,
refletindo progresso e compreensão dos fundamentos da linguagem.

## ⚙️ Como usar

1. Clone o repositório
2. Abra no IntelliJ IDEA (ou outra IDE)
3. Compile e execute as classes conforme necessidade
